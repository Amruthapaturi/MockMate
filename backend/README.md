# AI Interview Practice - Express Backend

## Setup Instructions

1. Create a new folder called `backend`
2. Copy all files from this folder into your backend directory
3. Run the following commands:

```bash
cd backend
npm init -y
npm install express cors
node server.js
```

The backend will run on `http://localhost:5000`

## Files to Create

### 1. server.js
### 2. routes/auth.js
### 3. routes/interview.js
### 4. data/questions.js
### 5. utils/evaluator.js

See the individual files in the `backend/` folder.
